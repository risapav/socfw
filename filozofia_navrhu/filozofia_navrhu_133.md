## Commit 221 — artifact-aware `files.tcl` emitter fix

```text
emit: preserve QIP SDC and RTL artifacts in native files.tcl
```

Cieľ:

* opraviť stratený `.qip`
* rozlíšiť `.sv/.v`, `.qip`, `.sdc`
* pridať test, ktorý zabráni regresii

## Oprava v `FilesTclEmitter`

Logika má byť:

```python
def _assignment_for(path: str) -> str | None:
    lower = path.lower()

    if lower.endswith(".qip"):
        return f'set_global_assignment -name QIP_FILE "{path}"'

    if lower.endswith(".sdc"):
        return f'set_global_assignment -name SDC_FILE "{path}"'

    if lower.endswith(".sv"):
        return f'set_global_assignment -name SYSTEMVERILOG_FILE "{path}"'

    if lower.endswith(".v"):
        return f'set_global_assignment -name VERILOG_FILE "{path}"'

    return None
```

A emitter nesmie robiť toto:

```python
if fp.endswith(".qip"):
    continue
```

ak zároveň nemá osobitnú QIP sekciu.

## Test

```python
def test_files_tcl_preserves_qip_sdc_and_rtl(tmp_path):
    ...
    text = Path(out).read_text()

    assert 'QIP_FILE' in text
    assert 'clkpll.qip' in text
    assert 'SDC_FILE' in text
    assert 'clkpll.sdc' in text
    assert 'SYSTEMVERILOG_FILE' in text or 'VERILOG_FILE' in text
```
